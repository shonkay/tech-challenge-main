# Technical Challenge

## Tools

- Reactjs
- SCSS

## Getting Started

- Clone the project
- cd into the project folder
- Run <code>npm install</code>
- Run <code>npm start</code>
- You can view on the browser using [http://localhost:3000](http://localhost:3000)

## Available Route

- / For search functionality
- /people
- /categories
- /categories/{categoryName}
